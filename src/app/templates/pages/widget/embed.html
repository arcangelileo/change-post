Embeddable Widget

Add a "What's New" notification to your app

Installation

Add this script tag to your website's HTML, just before the closing </body> tag.

<script src="{{ base_url }}/api/widget/{{ project.slug }}/embed.js" async></script>

What it looks like

Floating bell button

A notification bell appears in the bottom-right corner of your page. A red badge shows the count of new updates.

Popup panel

Clicking the bell opens a panel showing your latest published changelog posts with titles, categories, and excerpts.

Links to your changelog

Each post links to its full detail page on your public changelog.

Features

  • Uses your project's brand color ( {{ project.accent_color }})
  • Tracks "new since last viewed" with localStorage
  • Lightweight vanilla JS — no dependencies
  • Mobile responsive — full-width on small screens
  • Self-contained — all styles inline, no CSS conflicts
  • Shows last 5 published posts with category badges

JSON API

Prefer building your own widget? Use the JSON API to fetch recent posts.

GET {{ base_url }}/api/widget/{{ project.slug }}/posts?limit=5
